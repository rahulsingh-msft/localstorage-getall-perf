LocalStorage GetAll() Cold Read Benchmark

Measures the synchronous GetAll() Mojo IPC that fires on the first localStorage access after browser restart. Compare SQLite vs LevelDB backends.

Cold-read workflow:
  1. Populate — write test entries to localStorage
  2. Close the browser — flushes all data to disk and tears down in-memory caches
  3. Reopen the browser and navigate back to this page
  4. Measure — click the button to time the cold GetAll() from disk
Tip: append ?auto to the URL to measure automatically on page load (useful for scripted multi-restart runs).
